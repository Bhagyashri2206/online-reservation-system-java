# Online Reservation System (Java)

## Project Description
This project is a Java-based Online Reservation System that allows users to log in,
reserve tickets, and cancel reservations. The system stores all data in a MySQL
database using JDBC.

## Technologies Used
- Java
- JDBC
- MySQL

## Modules
- Login Authentication
- Ticket Reservation
- Ticket Cancellation

## Database Tables
- users
- trains
- reservations

## How to Run
1. Create MySQL database using provided SQL scripts
2. Update database credentials in DBConnection.java
3. Run Main.java
